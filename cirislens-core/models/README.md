# Scrubbing v2 model artifacts

Generated by `scripts/export_xlmr_ner.py`. Not committed to git (~280 MB INT8).

To produce the artifacts:
```bash
python -m venv /tmp/scrub_export_venv
source /tmp/scrub_export_venv/bin/activate
pip install transformers 'optimum[onnxruntime]' onnxruntime
python scripts/export_xlmr_ner.py
```

Output:
```
xlmr-ner.onnx                INT8-quantized XLM-R NER (~280 MB)
tokenizer.json               Hugging Face tokenizer config
sentencepiece.bpe.model      XLM-R subword vocabulary
labels.txt                   Comma-separated BIO label list
```

To enable NER inference at runtime, build with `--features ner` and set:
```
CIRISLENS_NER_MODEL_PATH=cirislens-core/models/xlmr-ner.onnx
CIRISLENS_NER_TOKENIZER_PATH=cirislens-core/models/tokenizer.json
CIRISLENS_NER_LABELS=$(cat cirislens-core/models/labels.txt)
```

See `FSD/CIRIS_SCRUBBING_V2.md` § Critical path Stage 1 for the full design.
